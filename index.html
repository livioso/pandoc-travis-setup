Pandoc Travis Setup

Build PDFs from Markdown on TravisCI.

Purpose

Showcase for an automated process of building PDFs from Markdown files using Travis + Pandoc. The example document get compiled and automatically committed to the gh-pages branch. The compiled PDF is stored in dist/.
